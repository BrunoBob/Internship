This project was made by Bruno Tessier during a 6 month Internship at ishiguro Laboratory in 2018.

The depository is organized in different folder : 

	- Biblio : Contain some paper related to the work realised

	- Data : Folder to put the motion database file

	- ProcessData : Contain the scripts to process the data 

	- Recording : Contain all necessary file to make a recording of face motion with a motion capture camera

The different folder may contain a Readme file to describe how to use its content.
